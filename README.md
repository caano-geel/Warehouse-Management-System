\# Warehouse Management System (WMS)



A web-based Warehouse Management System built with PHP, CodeIgniter, MySQL, Bootstrap, and JavaScript.



\## Live Demo



https://warehouse.freedev.app/



\## Features



\- Dashboard Analytics

\- Goods Management

\- Supplier Management

\- Customer Management

\- Incoming \& Outgoing Goods Tracking

\- Stock Monitoring

\- PDF Reports

\- User Management



\## Local Setup



1\. Copy project to XAMPP `htdocs`.

2\. Create database `wmsci`.

3\. Import `DATABASE FILE/wmsci.sql`.

4\. Configure `application/config/database.php`.

5\. Open `http://localhost/wms/`.



\## Production Setup



1\. Upload files to hosting.

2\. Create MySQL database.

3\. Import SQL file.

4\. Update `base\_url` and database credentials.
